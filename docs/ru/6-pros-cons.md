# ЗАЧЕМ ЭТО ВООБЩЕ?

## Преимущества

### 1. Ты — бот

Юзербот сидит под твоим аккаунтом. Не надо создавать отдельного бота, настраивать его, приглашать в комнаты. Все твои комнаты доступны сразу.

### 2. Минимум внешних зависимостей

Всё, что нужно — Python + доступ к Matrix серверу. Не нужны:
- Отдельный сервер для бота
- Сложной настройки

Фактически: поставил UV, склонировал репу, запустил — работает.

### 3. Встроенный репозиторий модулей

По умолчанию подключён системный репозиторий: `https://raw.githubusercontent.com/MxUserBot/mx-modules/main`

Любой модуль оттуда ставится одной командой: `.mdl module_name`
Можно добавить свои репозитории: `.addrepo https://github.com/user/repo`


### 4. Эмодзи-коллбеки вместо кнопок

Реакции как кнопки, работает везде, почему нет? Раз нет инлайн кнопок.

### 5. Веб-панель

FastAPI веб-морда для первого логина, управления модулями и конфигом. Не надо лезть в консоль.

### 6. Безопасность community кода

Community модули работают в песочнице: AST-анализ при загрузке, audit hook на file/socket/import, ScopedDatabase, frozen core-модули. Вредоносный модуль не сольёт ключи и не выполнит shell.

### 7. SAS верификация

Верификация устройств прямо из бота, без сторонних утилит. Полный цикл: request → emoji → MAC → done.
